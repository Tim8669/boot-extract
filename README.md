Simple tool for extracting kernel, ramdisk and metadata from an
Android boot or recovery image.

## To build (on Linux):  
```
  $ make
```
## Usage:
```
  $ ./boot-extract <-i> [boot or recovery image file]
                    -i   information only: do not extract images
```
